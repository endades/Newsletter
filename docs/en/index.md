# What is the purpose of a Version Newsletter?

The Version Newsletter aims to **inform users and technicians** about improvements, fixes, and new features included in each software update.

---

# Why is it important?

- **Keeps everyone informed:**  
  Users and technical teams are aware of the latest available updates.

- **Increases transparency:**  
  Clearly communicates the implemented changes and the value they provide.

- **Facilitates version management:**  
  IT managers can plan deployments and updates in an organized way.

- **Improves user experience:**  
  By knowing whatâ€™s new, users can make better use of newly added features.
